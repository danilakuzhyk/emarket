### High level details of what going on

## As a customer

* I visit app in browser
* I can login/register/drop password
* After i login i redirected to page where i can see some goods and can filter them
* I should have an ability to visit profile page where i can see my shipment addresses, billing address, wallet info.
* On main page i should be able to select goods that i want to buy to go to detailed page
* On detailed page i should be able to store goods in basket
* I can freely visit my basket from main page, where i see all goods that i've stored
* I can freely chose any goods that im able to buy now (means i want to buy them) click button and make an order
* After i make order total sum of goods should be freezen on my wallet (means money reserved for this orders), if i have no money, i should deposit some
* After my orders where approved (means vendor can actually sell them to me), money from my wallet are gone
* After everything was sold to me i should be able to see shipment status for goods that i've bought (on warehouse, in transit, received etc)

## As a vendor

* I visit app in browser
* I can login/register/drop password
* After i login i redirected to page where i can see my goods and can filter them and can add new
* I should have an ability to visit profile page where i can see quantity, price etc and can change those.
* I should have an ability to see incoming order request. Check them and approve
* I should be abdle to manually set status of order so that user new where is his stuff right now
