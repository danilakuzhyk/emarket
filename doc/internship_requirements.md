## This doc declares requirements for new rust interns

### Required:
* Basic rust/c++/c#/c/go knowledge
* Basic REST knowledge
* Basic SQL databases knowledge

### Nice-To-Have:
* One(Actix, Axum, etc) rust web api framework knowledge
* Jwt authorization
* Basic microservices knowledge. Pros and Cons. Message brokers
* Basic blockchain understanding
* One(react, angular, vue, htmx) frontend knowledge
* Basic rust-to-sql knowledge
* One rust ORM basic knowledge
